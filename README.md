# iOS Intensive Labs

A collection of focused practice projects and "freezer" demos for mastering core iOS development concepts.

## Modules

-   🧵 **GCD (Grand Central Dispatch):** Demonstrates background thread execution vs. main thread UI freezing.
-   ⏳ **Concurrency:** Practice with modern Swift `async/await`, `Task`, and `@MainActor`.
-   ♻️ **Lifecycle VC:** Deep dive into `UIViewController` lifecycle methods and background task persistence.
-   🧠 **Memory (ARC):** Identifying and fixing memory leaks (retain cycles) using `[weak self]`.
-   🌐 **Networking:** Basic API integration using `URLSession` and `Codable`.
-   🎨 **SwiftUI & UIKit:** Programmatic UI construction and SwiftUI-UIKit interoperability via `UIHostingController`.
